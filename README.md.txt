# Procurement Projects Dashboard (GitHub Pages)

A simple single-page dashboard to manage projects → sections/categories → procurement log items.

## Features
- Multiple projects (Project Name + Ref No required; others optional)
- Section/category management (auto color)
- Add/Edit/Delete items
- Filters: section + approval + search + sorting
- Excel import/export (XLSX)
- PDF export
- In-app notifications + optional browser notifications

## Deploy on GitHub Pages
1. Create a repo (example: `procurement-dashboard`)
2. Add: `index.html`, `styles.css`, `app.js`
3. Settings → Pages → Deploy from branch → `main` / root

## Notes
- Data saves locally in your browser (LocalStorage).
- To share data across devices, export Excel and import it on the other device.
